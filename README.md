# Kite Group Pty Ltd Website

Static marketing website for Kite Group Pty Ltd.

## Files

- `index.html`: main website
- `styles.css`: responsive styling
- `script.js`: section reveal animation
- `netlify.toml`: Netlify deploy config
- `thank-you.html`: form success page

## Still needed before going live

- business phone number
- business email address
- real logo and project photos if available
- final confirmation of service areas if `QLD all areas` and `NT all areas` are correct

## Deploy on Netlify

1. Create a GitHub repository and upload these files.
2. Log in to Netlify.
3. Click `Add new site` -> `Import an existing project`.
4. Connect the GitHub repository.
5. Deploy settings:
   - Build command: leave empty
   - Publish directory: `.`
6. Netlify will host the site and process the enquiry form submissions.

## Custom domain

1. Buy or use an existing domain such as `kitegroup.com.au`.
2. In Netlify, open `Domain management`.
3. Add the custom domain and follow the DNS instructions from your domain provider.
